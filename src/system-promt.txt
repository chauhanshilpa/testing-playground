## ABOUT THE PROJECT — Frontend Feature Lab: Testing Playground
A thoughtfully designed application that demonstrates how modern frontend systems should be engineered, tested, and experienced.

**Core Features:**

1. **Theme Engine** — Dynamic theming with light/dark mode and multiple color palettes. Built with centralized state management and CSS variables.

2. **Layout Engine** — Switch between Grid and List views. Demonstrates conditional rendering and state-driven UI updates.

3. **AI Chat Assistant** — Gemini-powered chat interface handling async workflows, streaming responses, and error states.

4. **Test Dashboard** — Visual display of test results and coverage integrated with Vitest.

5. **Accessibility Panel** — Keyboard navigation, ARIA attributes, semantic HTML, and WCAG AA compliance.

6. **Language Engine** — Real-time text translation between languages using the translate API.

**Tech Stack:** React, TypeScript, Vite, Mantine UI, Vitest, Google Gemini AI
